New in this release

Features

  • AppMap side-bar panel

    Find all your AppMaps in a single view. The AppMap panel displays a flat list of every AppMap in the workspace. AppMaps are sorted alphabetically by name.

  • Command: Find an AppMap by name

    Search by name to locate and open a single AppMap in your current workspace.

  • Command: Filter all AppMaps by name

    Filter all the AppMaps in your current workspace. Results appear in the AppMap panel.

  • Command: Focus on local files view

    Opens the AppMap panel. This is the same as clicking the AppMap logo in the side-bar.

  • Event preview

    Click the eye icon within an event list item to preview the event in the Trace view. The view will shift focus to the event, without changing the active selection.

Bug fixes

  • HTTP response status codes are now properly displayed in the Trace view for Python applications
  • Fixed an issue that would result in some lesser-used SQL keywords to disappear.
